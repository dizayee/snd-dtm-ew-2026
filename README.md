## Recorded Events Triggering Displacement in Sudan

Monthly incidents by type from April 2023 to April 2026, tracking three categories: armed conflict, fire, and rains & flooding.

Data is loaded from a CSV and rendered as overlapping line and area charts using D3.js. Each data point is marked with a dot. Numbered event annotations below the chart correspond to key incidents with titles and descriptions pulled from the same CSV.

Made with [d3.annotation()](http://d3-annotation.susielu.com/).
